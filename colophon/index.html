Colophon

How this site is made, and why it is made this way.

No framework, no build

This site is plain HTML, one stylesheet, and one small JavaScript file, written by hand. There is no framework, no bundler, no preprocessor, no npm — nothing to install and nothing to compile. What is in the repository is exactly what your browser receives. Each page is a standalone document; the shared header and footer are duplicated across pages on purpose, because at five pages, duplication is simpler than any machinery for avoiding it.

The one piece of JavaScript — the clock in the footer — is typed but never compiled: plain JS with // @ts-check and JSDoc annotations, so the editor checks types while the browser runs the file as-is. Every page works fully with JavaScript disabled; scripts only upgrade what is already there.

Typography

The site is all text, so the type does all the work. Everything is set in Literata, a serif designed for long-form reading on screens, used here as a single variable font with optical sizing — the same file supplies text and display cuts. It is self-hosted as a latin-subset woff2, licensed under the SIL Open Font License, with a serif system stack standing in while it loads. The scale is fluid, the measure is held to a readable 65–75 characters, and there are no photographs, no illustrations, and no icons anywhere.

Color & theming

Colors live as design tokens at the top of the stylesheet: warm off-white paper, near-black ink, and a khaki-and-olive accent family, with every text color meeting WCAG AA contrast. Dark mode follows your system preference and is implemented entirely by re-assigning the tokens — the accent shifts toward a desaturated sage rather than a muddy olive. Animation respects your reduced-motion preference, which is easy to do because there is almost none of it.

The clock

The footer tells the time in literature: a small custom element rounds the clock to the nearest quarter-hour and quotes a book that mentions it. Without JavaScript it still tells you something true about time, via C.S. Lewis. It has its own case study.

Analytics

Visits are counted with GoatCounter, chosen because it is open source and counts visits without cookies, without profiles, and without following you to other sites — a tally, not a tracker. Its script is the only third-party request this site makes.

Hosting

The site is static files served by GitHub Pages, deployed by pushing to the repository. Previewing locally takes one line of Python and no dependencies: python3 -m http.server.
